
# mobile_arquitetura_01

Aplicação Flutter que consome a API pública FakeStore.

## Arquitetura

- presentation → interface do usuário
- domain → entidades e contratos
- data → models, datasources e repositórios
- core → utilitários

## API
https://fakestoreapi.com/products

## Funcionalidades

- Consumo de API REST
- Conversão de JSON para objetos
- Listagem de produtos com título, preço e imagem

## Executar

flutter pub get
flutter run
